This is a program that calculates perfect numbers

A perfect number is a number in which all it's factors add up to the number itself.
For example, the factors for the number 6 are 1, 2 and 3. 1 + 2 + 3 = 6, so 6 would be perfect.
You can use this site to quickly find out if a number if perfect or not.
Type in any number into the text box. Click the button to find out whether it's perfect or not.
